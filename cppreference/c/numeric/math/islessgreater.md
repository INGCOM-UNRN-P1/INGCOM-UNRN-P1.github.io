#define islessgreater(x, y) /* implementation defined */ |  |  (since C99)  
| |   
  
Determines if the floating-point number x is less than or greater than the floating-point number y, without setting floating-point exceptions.

### [[edit](https://en.cppreference.com/mwiki/index.php?title=c/numeric/math/islessgreater&action=edit&section=1 "Edit section: Parameters")] Parameters

---|---|---  
y  |  \-  |  floating-point value   
  
### [[edit](https://en.cppreference.com/mwiki/index.php?title=c/numeric/math/islessgreater&action=edit&section=2 "Edit section: Return value")] Return value

Nonzero integral value if x < y || x > y, ​0​ otherwise. 

### [[edit](https://en.cppreference.com/mwiki/index.php?title=c/numeric/math/islessgreater&action=edit&section=3 "Edit section: Notes")] Notes

The built-in operator< and operator> for floating-point numbers may raise [FE_INVALID](../fenv/FE_exceptions.html "c/numeric/fenv/FE exceptions") if one or both of the arguments is NaN. This function is a "quiet" version of the expression x < y || x > y. The macro does not evaluate x and y twice. 

### [[edit](https://en.cppreference.com/mwiki/index.php?title=c/numeric/math/islessgreater&action=edit&section=4 "Edit section: Example")] Example

Run this code
    
    
    #include <math.h>
    #include <stdio.h>
     
    int main(void)
    {
        [printf](../../io/fprintf.html)("islessgreater(2.0,1.0)      = %d\n", islessgreater(2.0, 1.0));
        [printf](../../io/fprintf.html)("islessgreater(1.0,2.0)      = %d\n", islessgreater(1.0, 2.0));
        [printf](../../io/fprintf.html)("islessgreater(1.0,1.0)      = %d\n", islessgreater(1.0, 1.0));
        [printf](../../io/fprintf.html)("islessgreater(INFINITY,1.0) = %d\n", islessgreater(INFINITY, 1.0));
        [printf](../../io/fprintf.html)("islessgreater(1.0,NAN)      = %d\n", islessgreater(1.0, NAN));
     
        return 0;
    }

Possible output: 
    
    
    islessgreater(2.0,1.0)      = 1
    islessgreater(1.0,2.0)      = 1
    islessgreater(1.0,1.0)      = 0
    islessgreater(INFINITY,1.0) = 1
    islessgreater(1.0,NAN)      = 0

1-3) Decomposes given floating-point value arg into integral and fractional parts, each having the same type and sign as arg. The integral part (in floating-point format) is stored in the object pointed to by iptr.

### [[edit](https://en.cppreference.com/mwiki/index.php?title=c/numeric/math/modf&action=edit&section=1 "Edit section: Parameters")] Parameters

arg  |  \-  |  floating-point value   
---|---|---  
iptr  |  \-  |  pointer to floating-point value to store the integral part to   
  
### [[edit](https://en.cppreference.com/mwiki/index.php?title=c/numeric/math/modf&action=edit&section=2 "Edit section: Return value")] Return value

If no errors occur, returns the fractional part of arg with the same sign as arg. The integral part is put into the value pointed to by iptr. 

The sum of the returned value and the value stored in *iptr gives arg (allowing for rounding). 

### [[edit](https://en.cppreference.com/mwiki/index.php?title=c/numeric/math/modf&action=edit&section=3 "Edit section: Error handling")] Error handling

This function is not subject to any errors specified in [`math_errhandling`](math_errhandling.html "c/numeric/math/math errhandling"). 

If the implementation supports IEEE floating-point arithmetic (IEC 60559), 

  * If arg is ±0, ±0 is returned, and ±0 is stored in *iptr. 
  * If arg is ±∞, ±0 is returned, and ±∞ is stored in *iptr. 
  * If arg is NaN, NaN is returned, and NaN is stored in *iptr. 
  * The returned value is exact, [the current rounding mode](../fenv/FE_round.html "c/numeric/fenv/FE round") is ignored. 



### [[edit](https://en.cppreference.com/mwiki/index.php?title=c/numeric/math/modf&action=edit&section=4 "Edit section: Notes")] Notes

This function behaves as if implemented as follows: 
    
    
    double modf(double value, double *iptr)
    {
    #pragma STDC FENV_ACCESS ON
        int save_round = [fegetround](../fenv/feround.html)();
        [fesetround](../fenv/feround.html)([FE_TOWARDZERO](../fenv/FE_round.html));
        *iptr = std::[nearbyint](nearbyint.html)(value);
        [fesetround](../fenv/feround.html)(save_round);
        return [copysign](copysign.html)([isinf](isinf.html)(value) ? 0.0 : value - (*iptr), value);
    }

### [[edit](https://en.cppreference.com/mwiki/index.php?title=c/numeric/math/modf&action=edit&section=5 "Edit section: Example")] Example

Run this code
    
    
    #include <float.h>
    #include <math.h>
    #include <stdio.h>
     
    int main(void)
    {
        double f = 123.45;
        [printf](../../io/fprintf.html)("Given the number %.2f or %a in hex,\n", f, f);
     
        double f3;
        double f2 = modf(f, &f3);
        [printf](../../io/fprintf.html)("modf() makes %.2f + %.2f\n", f3, f2);
     
        int i;
        f2 = [frexp](frexp.html)(f, &i);
        [printf](../../io/fprintf.html)("frexp() makes %f * 2^%d\n", f2, i);
     
        i = [ilogb](ilogb.html)(f);
        [printf](../../io/fprintf.html)("logb()/ilogb() make %f * %d^%d\n", f / [scalbn](scalbn.html)(1.0, i), [FLT_RADIX](../../types/limits.html), i);
     
        // special values
        f2 = modf(-0.0, &f3);
        [printf](../../io/fprintf.html)("modf(-0) makes %.2f + %.2f\n", f3, f2);
        f2 = modf(-INFINITY, &f3);
        [printf](../../io/fprintf.html)("modf(-Inf) makes %.2f + %.2f\n", f3, f2);
    }

Possible output: 
    
    
    Given the number 123.45 or 0x1.edccccccccccdp+6 in hex,
    modf() makes 123.00 + 0.45
    frexp() makes 0.964453 * 2^7
    logb()/ilogb() make 1.92891 * 2^6
    modf(-0) makes -0.00 + -0.00
    modf(-Inf) makes -INF + -0.00

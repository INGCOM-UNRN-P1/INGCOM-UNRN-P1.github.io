This header defines macros for [alternative operator spellings](../language/operator_alternative.html "c/language/operator alternative"). 

### [[edit](https://en.cppreference.com/mwiki/index.php?title=c/header/iso646&action=edit&section=1 "Edit section: Synopsis")] Synopsis
    
    
    #define and     &&
    #define and_eq  &=
    #define bitand  &
    #define bitor   |
    #define compl   ~
    #define not     !
    #define not_eq  !=
    #define or      ||
    #define or_eq   |=
    #define xor     ^
    #define xor_eq  ^=

### [[edit](https://en.cppreference.com/mwiki/index.php?title=c/header/iso646&action=edit&section=2 "Edit section: Notes")] Notes

This header is meaningless in C++ because these macro names exist as keywords in C++ and are unconditionally available.
